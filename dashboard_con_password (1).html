📂 Cómo usar el Refresh con OneDrive

Opción A — Archivo local (solo tú):
1. Pulsa 📂 Cargar Excel y selecciona el archivo desde tu carpeta de OneDrive local
2. Pulsa 🔄 Refresh para recargar cuando haya cambios

Opción B — Enlace OneDrive (tú y usuarios compartidos):
1. En OneDrive web, haz clic derecho en el Excel → Copiar enlace
2. Pégalo en el campo de abajo, cambia el final e=xxxx por download=1
3. Pulsa ⬇️ Cargar — el enlace se guarda para siempre
4. Cuando el Excel se actualice en OneDrive, pulsa 🔄 Refresh
